Rapunzel

Rapunzel is the fs based note-taking app focuses markdown.

Three Goals

  • We can use this on browsers.
  • We can use this in offline.
  • We can use the file-system based on NodeJS.fs in this.

Tech Stack

Language Typescript
Library React, MobX, CodeMirror, Browserfs, Marked, office-ui-fabric
Bundler Webpack
Linter TSLint
Test Jest
